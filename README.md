# Learnzy — AI English Learning for Indian Kids

AI-powered English learning app for children aged 3–12 (LKG to Class 5).

## Features (V1)
- 🗣️ AI Voice Chat — 4 animated buddies (Cuty, Candy, Bunny, Jumbo), 6 subject modes
- 📖 Vocabulary — 1,500+ words by class level
- 🔤 Phonics — structured phonics curriculum
- 🔢 Maths Magic — 12 tricks from LKG to Class 5
- ✏️ Daily Exercises — AI-generated questions, refreshes every day
- 🎮 Game Zone — 15 educational games
- 📝 Grammar, Writing Workshop, Word Explorer

## V2 (coming)
- 🎬 Wonder Videos with Arya (50 videos across 5 STEM categories)
- 🧩 Brain Challenges

## Stack
- Frontend: Expo / React Native
- Backend: FastAPI + MongoDB
- AI: OpenAI (GPT-4o-mini, Whisper, TTS)
- Payments: Razorpay
